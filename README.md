## To run this project
First of run 
```bash
yarn
```
in the root and in the /studio folder.

Then do 
```bash
sanity init
```
and follow the instrucions to create a new Sanity project

Add the new project id where needed.

Lastly run
```bash
yarn dev
```
in the root folder.
